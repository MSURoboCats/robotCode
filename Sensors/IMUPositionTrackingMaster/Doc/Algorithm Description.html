Propagation

The state vector is quaternion qqq

Ft=I4+12dt∗Ω(ωt)F_t=I_4+\dfrac{1}{2}dt*\Omega(\omega_t) Ft​=I4​+21​dt∗Ω(ωt​)

Make estimation:

q=Ftqq=F_tq q=Ft​q

Measurement Update

Here we only use the unit vector of our measurements to avoid accumulation of noise errors.

ea=at∣∣at∣∣2ea=\dfrac{a_t}{||a_t||_2} ea=∣∣at​∣∣2​at​​

em=mt∣∣mt∣∣2em=\dfrac{m_t}{||m_t||_2} em=∣∣mt​∣∣2​mt​​

pm=Normalize(R(q)mn)pm=Normalize(R(q)m_n) pm=Normalize(R(q)mn​)

so error:
